YouTube Video Summarizer

How it Works

This app provides a summary of a YouTube video using OpenAI's Whisper API (speech-to-text transcription) and GPT-3.5 (for summarizing the text). To use this app, enter the link to the YouTube video and your OpenAI API key, and then click on "Preview Video". After previewing the video, click on "Continue" to start the transcription process. The summary will be displayed once the transcription is complete.

How to Sign Up for an OpenAI Account

  1. Go to the OpenAI website for API Keys (https://platform.openai.com/account/api-keys).
  2. Login/Create Account with OpenAI.
  3. Follow the instructions to create an account and obtain your API key.
